Ubuntu环境下安装最新版Node.js

Author: Cage Wang Date: December 26, 2022  15:41:53 Category: 环境配置

由于 Windows 环境下的 Node.js 稳定性较差，实验时容易报错，故尝试在 Windows 环境下使用 wsl2 构建 Ubuntu 环境进行实验。Node.js 官网只为 Linux 提供可执行的二进制文件，按照网上设置软链接的方式操作，在运行 Node 时会出现缺少文件的错误。而使用 apt install 方法安装的 Node 版本过旧，故尝试使用其他方法安装，以下纪录安装过程。
